# Emotion-Estimation
An initial release of Emotion Estimation

## Introduction ##
A real-time facial emotion estimation model.

## Dependencies ## 
-- Python >=2.7

## Start ##
Firstly, install libaries.  
```bash
$ pip install keras numpy cv2
```
Then.
```bash
$ cd Emotion-Estimation
```
Lastly.
```bash
$ python model.py
```
